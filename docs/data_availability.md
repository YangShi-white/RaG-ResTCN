# Data Availability

This repository does not redistribute raw fermentation or Raman spectral data.

The primary benchmark data should be obtained from:

Goldrick, Stephen (2019), "Data for: Modern day monitoring and control challenges outlined on an industrial-scale benchmark fermentation process", Mendeley Data, V1, doi: `10.17632/pdnjz7zz5x.1`.

The external Raman validation uses a separate Raman spectroscopy dataset associated with doi: `10.1002/bit.70211`.

After downloading the IndPenSim data, place the raw CSV files under `data/raw/` and run the preprocessing scripts listed in `README.md`.
